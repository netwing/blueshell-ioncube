MovieClip.lockTween() , TextField.lockTween()

Availability

Flash Player 6.

Usage

my_mc.lockTween()

Parameters

none

Returns

nothing

Description

method, disallows performing tweens on movieclip or textfield (useful with buttons)
to unlock use .unlockTween() method

The following tween call has no result since my_mc is locked :

my_mc.lockTween();
my_mc.tween(["_x","_y"],[0,0]);
