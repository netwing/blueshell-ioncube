MovieClip.brightOffsetTo(), TextField.brightOffsetTo()

Availability

Flash Player 6.

Usage

my_mc.brightOffsetTo(percent, seconds, animtype, delay, callback, extra1, extra2)

Parameters

percent A number that represents the brightOffset of movieclip or textfield
0 : normal
100 : to white
-100 : to black

Other parameters follow the tween() method.

Returns

Nothing.

Description

Following example changes tint of my_mc to brightOffset 100 in 2 seconds

my_mc.brightOffsetTo(100,2,"easeOutSine");
